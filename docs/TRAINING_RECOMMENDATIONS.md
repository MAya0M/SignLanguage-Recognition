# המלצות לאימון המודל

## שאלה 1: כמה סרטונים לכל מילה צריך?

### המצב הנוכחי שלך:
- **יש לך כרגע: 16-17 סרטונים לכל מילה** ✅ (לא רע!)
- **סך הכל: 129 סרטונים, 8 מילים**

### המלצות לפי רמות:

#### 🟡 מינימום (לעבודה בסיסית):
- **10-15 סרטונים לכל מילה**
- מספיק לבדיקות ופיתוח
- דיוק: ~60-70%

#### 🟢 מומלץ (לשימוש טוב):
- **20-30 סרטונים לכל מילה**
- מספיק לשימוש מעשי
- דיוק: ~75-85%
- **זה מה שכדאי לך להוסיף!**

#### 🔵 אידיאלי (לשימוש מקצועי):
- **50+ סרטונים לכל מילה**
- דיוק: ~85-95%
- מתאים למוצר מסחרי

### מה כדאי להוסיף?

**כרגע יש לך 16-17 סרטונים לכל מילה**

**מומלץ להוסיף עוד 5-15 סרטונים לכל מילה** כדי להגיע ל-20-30 סה"כ.

**סה"כ צריך להוסיף: 40-120 סרטונים נוספים** (5-15 לכל מילה × 8 מילים)

### טיפים להקלטת סרטונים טובים:

1. **גיוון באנשים** - הקלט אנשים שונים (אם אפשר)
2. **גיוון בזוויות** - זוויות שונות של המצלמה
3. **גיוון בתאורה** - תנאי תאורה שונים
4. **גיוון במיקום** - רקעים שונים
5. **גיוון במהירות** - אנשים שמדברים בקצב שונה
6. **גיוון בסגנון** - וריאציות שונות של אותה מילה

## שאלה 2: האם צריך לאמן מחדש את המודל?

### ✅ **לא צריך לאמן מחדש!**

העדכונים שעשיתי הם **רק בקוד ה-inference** (זיהוי), ולא שינו:
- ❌ את המבנה של המודל
- ❌ את אופן האימון
- ❌ את הפורמט של המודל

### מה השתנה?

**רק הקוד שמשתמש במודל:**
- ✅ עכשיו מזהה כל מילה בנפרד (במקום רק מילה אחת)
- ✅ מחלק את הסרטון לקטעים
- ✅ מחזיר רשימה של כל המילים

**המודל עצמו נשאר אותו דבר!**

### מתי כן צריך לאמן מחדש?

1. **אם הוספת סרטונים חדשים** - אז כן, כדאי לאמן מחדש עם כל הנתונים
2. **אם רוצה שיפור בביצועים** - יותר נתונים = מודל טוב יותר
3. **אם הוספת מילים חדשות** - חייב לאמן מחדש

### איך לבדוק שהכל עובד?

1. **הפעל את האפליקציה:**
   ```bash
   python app.py
   ```

2. **נסה להעלות סרטון** - אמור לזהות כל מילה בנפרד

3. **נסה בלייב** - אמור לזהות כל מילה בזמן אמת

אם זה עובד - **הכל תקין, לא צריך לאמן מחדש!** ✅

## איך להוסיף סרטונים?

1. **הקלט סרטונים חדשים** - שמור אותם ב-`Data/rawVideos/[WORD_NAME]/`
2. **חלץ keypoints:**
   ```bash
   python extract_keypoints.py
   ```
3. **עדכן את ה-CSV:**
   ```bash
   python scripts/create_dataset_csv.py
   ```
4. **אמן מחדש:**
   - ב-Colab: הרץ את הנוטבוק מחדש
   - או מקומית: `python scripts/train_model.py ...`
